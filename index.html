Creating Accessible Courses in Canvas

A Practical Guide for Instructors

1
Step 1
Adopt an Accessibility Mindset
Accessibility is a legal requirement and benefits every student. Treat it as an ongoing practice, not a one-time task.
Required by law Benefits all students Ongoing process
2
Step 2 — Plan & Prioritize
A New Course
  • Start with the University Canvas Template
  • Create accessible content from the start
  • Export PDFs properly (Save As, not Print)
  • Link to PDFs — don't upload them
  • Use Ally to check as you build
B Existing Course
New Semester
  • Start fresh with Canvas Template, or
  • Smart Copy content you'll reuse
  • Run Ally Course Report
  • Prioritize: core docs → quick wins → complex
  • Update content before adding to Canvas
Mid-Semester
  • Remove unused files with TidyUp
  • Run Ally Course Report
  • Prioritize: core docs → quick wins → complex
3
Step 3
Apply Accessibility Guidelines
Follow these standards when creating or updating any content in Canvas.
Headings & structure Alt text Color contrast Descriptive links Accessible tables Captions & transcripts

Step 4
Address Common Accessibility Issues
Work through issues identified in your Ally Course Report. Tackle the most impactful items first.
